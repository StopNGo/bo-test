### Quick Start

1. Clone this repo:

   `git clone https://github.com/StopNGo/bo-test`

2. Install all packages:

   `npm i`

3. Run project in a development mode:

   `npm start`

4. Open your browser with the next addresses:

   `http://localhost:8080/` - client

   `http://localhost:5000/` - API server
